# Spring Batch Example

## How to Build
```bash
$ git clone https://github.com/leadpony/examples-java.git
$ cd examples-java/spring-boot-and-batch
$ mvn clean package
```

## How to Run
```bash
$ cd target
$ tar xzvf spring-boot-and-batch-1.0.0.tar.gz
$ cd spring-boot-and-batch-1.0.0
$ ./run.sh
```
